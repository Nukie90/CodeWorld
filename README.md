# CodeWorld

## backend
To start backend
    ```cd backend```

create venv
    Windows
    ```python -m venv env```
    macOS/Linux
    ```python3 -m venv env```

Activate venv 
    Windows
    ```env\Scripts\activate```
    macOS/Linux
    ```source env/bin/activate```

install dependencies
```pip install -r requirements.txt```

run the server
```fastapi dev main.py```
