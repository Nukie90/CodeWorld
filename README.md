## Backend

# Install ENV
1. cd backend
2. python -m venv venv || python3 -m venv venv

# To activate ENV
1. cd backend
2. source venv/bin/activate
3. pip3 install -r requirements.txt

# Checking
1. cd backend
2. uvicorn app.main:app --reload
